TITLE: What This Is Not

This project does not claim exclusive truth.

It does not ask for belief.
It does not promote extremism, ideology, or division.

The purpose is investigation.

If a model does not align with evidence, it must be refined or rejected.

The Codex encourages:
– Independent thinking
– Cross-checking sources
– Historical and scientific comparison
– Open evaluation
